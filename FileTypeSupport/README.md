# Trados Studio File Type Support Samples

This repository contains sample projects demonstrating how to extend Trados Studio's file type support using the File Type Support Framework.

## Projects

| Project | Description |
|---------|-------------|
| **Sdl.Sdk.FileTypeSupport.Samples.Bil** | Sample bilingual file type plugin with parser, writer, and sniffer components |
| **Sdl.Sdk.FileTypeSupport.Samples.SimpleText** | Complete file type implementation for simple text files, including preview support and user settings |
| **Sdl.Sdk.FileTypeSupport.Samples.SimpleText.WinUI** | WinUI components for the SimpleText file type settings interface |
| **Sdl.Sdk.FileTypeSupport.Samples.SimpleTextExtension** | Extension sample for the SimpleText file type |
| **Sdl.Sdk.FileTypeSupport.Samples.WordArtVerifier** | Verification plugin sample for checking Word Art elements |
| **Sdl.Sdk.FileTypeSupport.Samples.XMLChecker** | XML validation verifier plugin sample |

## Requirements

- .NET Framework 4.8
- Trados Studio instlalled

## Getting Started

1. Open `Sdl.Sdk.Samples.sln` in Visual Studio
2. Restore NuGet packages
3. Build the solution
